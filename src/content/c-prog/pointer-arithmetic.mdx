# Pointer Arithmetic

In C, when you add or subtract an integer from a pointer, it moves that many elements in memoryâ€”**not** bytes. For example, if `p` is an `int*` pointing at `0x1000`, then `p + 1` points at `0x1004` on a system where `sizeof(int) == 4`.

## Code Example

<CodeExample code={`
#include <stdio.h>

int main() {
    int arr[3] = {10, 20, 30};
    int *p = arr;            // p == &arr[0]
    printf("%d\\n", *(p+1)); // prints 20
    return 0;
}
